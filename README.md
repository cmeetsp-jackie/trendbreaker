# 🔮 TrendBreaker

**친근하고 신비로운 트렌드 분석 봇**

---

## 🎯 사용법

### **방법 1: 수동 분석 (가장 빠름)**

1. 형이 재주한테 슬랙으로: `"트렌드 분석: [키워드]"`
2. 재주가 즉시 분석해서 트윗 포맷으로 답변
3. 형이 만족하면: `"트윗해줘"`
4. 재주가 트위터에 발행!

**예시:**
```
형: "트렌드 분석: 손흥민 해트트릭"

재주: 
🔮 지금 이 순간, 한국 인터넷이 들썩이는 이유

#손흥민 해트트릭

💫 무슨 일이냐면요...
[분석 내용]

✨ 왜 중요한가?
[핵심]

🌙 #트렌드브레이커

형: "트윗해줘"

재주: ✅ 트윗 발행 완료!
🔗 https://twitter.com/xxx
```

---

### **방법 2: 네이버 트렌드 자동 크롤링**

```bash
node naver-crawler.js
```

- 네이버 쇼핑 인기 검색어 자동 수집
- 형한테 슬랙으로 알림
- 형이 원하는 키워드 선택해서 분석 요청

---

### **방법 3: 트위터 유료 API ($100/월)**

- 트위터 Basic 또는 Pro 구독 시
- 자동으로 트위터 트렌딩 키워드 크롤링 가능
- 1시간마다 자동 체크
